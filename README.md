## allPay歐付寶 全方位(All In One)金流介接 .Net版

### 1. 導覽

* allPay歐付寶金流服務對於有收款需求的賣家會員，提供完整的交易介接技術，並提供多種收款方式(All In One)選擇，
包含信用卡、網路ATM、ATM櫃員機、超商代碼、財付通、allPay儲值消費等，
本套件為.Net版，可以使用作建立訂單，接受付款通知，查詢訂單等金流交易應用。

### 2. 安裝環境

* .NET Framework3.5以上
